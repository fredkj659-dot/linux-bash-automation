## Description

Este proyecto automatiza la creación de un espacio de trabajo en Linux usando Bash.

El script principal crea carpetas necesarias para proyectos DevOps y muestra mensajes en consola confirmando la ejecución.

## Script

scripts/setup_workspace.sh

Este script realiza:

- Creación de estructura de carpetas
- Confirmación de ejecución en consola
- Preparación de entorno de trabajo

## How to Run

Dar permisos al script:

```bash
chmod +x scripts/setup_workspace.sh
